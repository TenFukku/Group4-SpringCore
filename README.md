# 📚 Ứng dụng Quản lý Sinh viên – Spring Boot & Hibernate

Đây là một dự án **Spring ORM** được xây dựng bằng **Spring Boot**, dùng để thực hiện các thao tác **CRUD (Create, Read, Update, Delete)** cơ bản trong quản lý sinh viên.

Mục tiêu của dự án là **demo và thực hành** các khái niệm cốt lõi trong việc tích hợp **Spring Framework với Hibernate** thông qua **Spring Data JPA**.

---

## 🛠️ Công nghệ sử dụng

- **Java 21**
- **Spring Boot 3.5.3** – Nền tảng chính để xây dựng ứng dụng
- **Spring Web** – Xây dựng các RESTful API
- **Spring Data JPA** – Truy cập dữ liệu dễ dàng, tích hợp Hibernate làm ORM mặc định
- **H2 Database** – Cơ sở dữ liệu trong bộ nhớ (in-memory), tiện lợi để phát triển và demo
- **Maven** – Quản lý project và thư viện phụ thuộc
- **Lombok** – Giảm thiểu boilerplate code cho các model
